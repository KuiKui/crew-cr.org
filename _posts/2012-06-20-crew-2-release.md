title: Crew 2.0 release
---

This release includes :

* a new [API](https://github.com/pmsipilot/Crew/wiki/API) more respectful of REST principles,
* a new layout which helps you to focus on what you have to do,
* a comment status system to check what is already done,
* a hacked [PHP Markdown Extra](https://github.com/wolfie/php-markdown) in comments,
* a listing of all "behind the scene" git operations in admin section,
* a feature to open files in your editor during code reviews,
* bugs fixes.

The [wiki](https://github.com/pmsipilot/Crew/wiki) was also updated.

Thanks to all [contributors](https://github.com/pmsipilot/Crew/graphs/contributors) and [PMSIpilot dev crew](http://pmsipilot.org) to help us improve this product.
